Mon site web portfolio


J'ai eu l'idée lors de ma première année en informatique (B1) de créer un site web hébergé sur GitHub, sur lequel vous vous trouvez actuellement.
Il s'agit de mon tout premier projet de développement web, un projet auquel j'ai consacré beaucoup de temps pour le créer et le maintenir
à jour, afin que les visiteurs puissent avoir accès à l'ensemble de mes travaux considérés comme intéressants.

Pour plus de détails, le site est hébergé gratuitement sur GitHub et est composé de pages HTML qui utilisent un seul fichier CSS.
Toutes les images et vidéos importées sont simplement stockées dans un sous-dossier appelé "data". La réalisation de ce portfolio
me permet de mettre en pratique mes compétences en développement web et de vous les présenter. Je suis encore loin d'être un
développeur web professionnel, mais l'objectif est davantage de me familiariser avec l'univers
des développeurs web pour mieux les comprendre.
