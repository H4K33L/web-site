Assistant de création de personnage pour jeu de rôle (JdR) :


J'ai commencé à développer ce projet en utilisant Python, le projet demandant de traiter des informations et de leur donner une nouvelle forme.
Mais seulement traiter les données n'est pas suffisant, il fallait aussi une interface utilisateur pour faciliter les interactions, j'ai donc utilisé la
bibliothèque Tkinter de Python pour faire mon interface utilisateur.

Je me sers d'un fichier CSV pour stocker les données du jeu de rôle. La structure respecte un format particulier pour pouvoir
être prise en compte par le programme et ainsi prendre sa forme finale en tant que fichier texte (TXT).
Dans ce fichier texte, on retrouvera toutes les informations nécessaires à la création de notre personnage.

Ambition et l'évolution future du projet :

Pour l'instant le code est lourd et très peu pratique, l'objectif à long terme est de faire en sorte que l'interface utilisateur
soit générée en fonction du CSV. Pour faire simple, un personnage de JdR est formé de plusieurs choix déterminants : Classe, histoire étendue, etc.
et ces choix principaux seront détectés par le code en regardant le CSV. Le code, par la suite, formera l'interface utilisateur de la manière
la plus optimisée. Ce qui permettra deux choses : la première est de rendre le code moins lourd en évitant d'implémenter tous les boutons.
Bien évidemment, je suis en train de travailler sur cette deuxième version, qui devrait aboutir sous peu.
