Assistant de création de personage de JDR :


J'ai commencé a développer ce projet en utilisant Python, le projet demandant de traiter des information et de leur donner une nouvelle forme.
Mais seulement traiter les données n'est pas sufisant, il falait aussi une interface utilisateur pour faciliter les interactions, j'ai donc utilisé la
bibliothéque Tkinter de python pour faire mon interface utilisateur.

Je me sert d'un csv pour stocker les données du jeu de role, la structure respecte un format particulier pour pouvoir
étre pris en compte par le programe et ainsi prendre sa forme finale de txt ou finalement on retrouvera toutes
les informations néscesaire a la création de notre personnage.

Ambition et ce vers quoi va evoluer le projet :

Pour l'instant le code est lourd et très peut pratique, l'objectif à long terme est de faire en sorte que l'interface utilisateur
soit généré en fonction du csv, pour faire simple un personnage de jdr est formé de plusieurs choix determinants : Classe, histoir ext, ext
et ces chois principaux seront detecté par le code en regardant le csv, code qui par la suite formera l'interface utilisateur de la façon
le plus optimisé. Ce qui permetra deux chosses : la premiére est de rendre le code moin lourd en evitant d'implémenter tout les boutons.
Bien évidament je suis en train de travailler sur cette deuxiéme version, qui devrai aboutir sous peut.
